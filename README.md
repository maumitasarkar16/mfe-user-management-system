# mfe-user-management-system
React Microfrontend User Management System - GIC assignment

---------------------------------------------------------------------------------------------------------
 ===================================================To run the servers~~~~~~~
For UserList MFE- 
   npm run build , 
   npx serve dist -l 5001 , 
   http-server dist -p 5001 -c-1 --cors

For UserDetails MFE- 
   npm run build , 
   npx serve dist -l 5002 , 
   http-server dist -p 5002 -c-1 --cors   

shell - dashboard 
   npm run dev
